1 Напишите регулярку, которая найдет строки 'ahb', 'acb', 'aeb' в let str = 'ahb acb aeb aeeb adcb axeb' по шаблону: буква 'a', любой символ, буква 'b'.

Результат:

Решение: str.match(/a.b/g)

2 Напишите регулярку, которая найдет строки 'abba', 'adca', 'abea' в let str = 'aba aca aea abba adca abea' по шаблону: буква 'a', 2 любых символа, буква 'a'.

Результат:

Решение: str.match(/a..a/g)

3 Напишите регулярку, которая найдет строки 'abba' и 'abea' в let str = 'aba aca aea abba adca abea', не захватив 'adca'.

Результат:

Решение: str.match(/ab.a/g)

4 Дано: let str = 'aa aba abba abbba abca abea'. Напишите регулярку, которая найдет строки 'aba', 'abba', 'abbba' по шаблону: буква 'a', буква 'b' любое количество раз, буква 'a'.

Результат:

Решение: str.match(/ab+a/g)

5 Дано: let str = 'aa aba abba abbba abca abea'. Напишите регулярку, которая найдет строки 'aa', 'aba', 'abba', 'abbba' по шаблону: буква 'a', буква 'b' любое количество раз (в том числе ниодного раза), буква 'a'.

Результат:

Решение: str.match(/ab*a/g)

6 Дано: let str = 'aa aba abba abbba abca abea'. Напишите регулярку, которая найдет строки 'aa', 'aba' по шаблону: буква 'a', буква 'b' один раз или ниодного, буква 'a'.

Результат:

Решение: str.match(/ab?a/g)

7 Дано: let str = 'ab abab abab abababab abea'. Напишите регулярку, которая найдет строки по шаблону: строка 'ab' повторяется 1 или более раз.

Результат:

Решение: str.match(/(ab)+/g)

8 Дано: let str = 'a.a aba aea'. Напишите регулярку, которая найдет строку 'a.a', не захватив остальные.

Результат:

Решение: str.match(/a\.a/g)

9 Дано: let str = '2+3 223 2223'. Напишите регулярку, которая найдет строку '2+3', не захватив остальные.

Результат:

Решение: str.match(/2\+3/g)

10 Дано: let str = '23 2+3 2++3 2+++3 345 567'. Напишите регулярку, которая найдет строки '2+3', '2++3', '2+++3', не захватив остальные (+ может быть любое количество).

Результат:

Решение: str.match(/2(\+)+3/g)

11 Дано: let str = '23 2+3 2++3 2+++3 445 677'. Напишите регулярку, которая найдет строки '23', '2+3', '2++3', '2+++3', не захватив остальные.

Результат:

Решение: str.match(/2(\+)*3/g)

12 Дано: let str = '*+ *q+ *qq+ *qqq+ *qqq qqq+'.Напишите регулярку, которая найдет строки '*q+', '*qq+', '*qqq+', не захватив остальные.

Результат:

Решение: str.match(/\*q+\+/g)

13 Дано: let str = '[abc] {abc} abc (abc) [abc]'. Напишите регулярку, которая найдет строки в квадратных скобках и заменят их на '!'.

Результат:

Решение: str.replace(/\[...\]/g, '!')

14 Дано: let str = 'aa aba abba abbba abbbba abbbbba'; Напишите регулярку, которая найдет строки 'abba', 'abbba', 'abbbba' и только их.

Результат:

Решение: str.match(/ab{2,4}a/g, '!')

15 Дано: let str = 'aa aba abba abbba abbbba abbbbba'; Напишите регулярку, которая найдет строки вида 'aba', в которых 'b' встречается менее 3-х раз (включительно).

Результат:

Решение: str.match(/ab{1,3}a/g)

16 Дано: let str = 'aa aba abba abbba abbbba abbbbba'; Напишите регулярку, которая найдет строки вида 'aba', в которых 'b' встречается более 4-х раз (включительно).

Результат:

Решение: str.match(/ab{4,}a/g)

17 Дано: let str = 'aba accca azzza wwwwa'; Напишите регулярку, которая найдет все строки по краям которых стоят буквы 'a', и заменит каждую из них на '!'. Между буквами a может быть любой символ (кроме 'a').

Результат:

Решение: str.replace(/a.+?a/g, '!')

18 Дано: let str = 'a1a a2a a3a a4a a5a aba aca'; Напишите регулярку, которая найдет строки, в которых по краям стоят буквы 'a', а между ними одна цифра.

Результат:

Решение: str.match(/a\d{1}a/g)

19 Дано: let str = 'a1a a22a a333a a4444a a55555a aba aca'; Напишите регулярку, которая найдет строки, в которых по краям стоят буквы 'a', а между ними любое количество цифр.

Результат:

Решение: str.match(/a\d+a/g)

20 Дано: let str = 'aa a1a a22a a333a a4444a a55555a aba aca'; Напишите регулярку, которая найдет строки, в которых по краям стоят буквы 'a', а между ними любое количество цифр (в том числе и ноль цифр, то есть строка 'aa').

Результат:

Решение: str.match(/a\d*a/g)

21 Дано: let str = 'avb a1b a2b a3b a4b a5b abb acb'; Напишите регулярку, которая найдет строки следующего вида: по краям стоят буквы 'a' и 'b', а между ними - не число.

Результат:

Решение: str.match(/a\D+b/g)

22 Дано: let str = 'ave a#b a2b a$b a4b a5b a-b acb'; Напишите регулярку, которая найдет строки следующего вида: по краям стоят буквы 'a' и 'b', а между ними - не буква и не цифра.

Результат:

Решение: str.match(/a\W+b/g)

23 Дано: let str = 'ave a#a a2a a$a a4a a5a a-a aca'; Напишите регулярку, которая заменит все пробелы на '!'.

Результат:

Решение: str.replace(/\s/g, '!')
